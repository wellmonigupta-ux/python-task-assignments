**Introduction**

-This repository contains basic Python programs created while learning Python through an online course.
The programs are written for beginners to practice simple concepts like input, calculations, and strings.

**Basic Mathematical Operations-**

*File name: task1_basic_math_operations.py*

**What this program does:**

-Takes two numbers from the user

**Performs:**

-Addition
-Subtraction
-Multiplication
-Division

**Displays the result of each operation on the screen Concepts Used:**

-User input (input())
-Number conversion (int() / float())
-Arithmetic operators (+ - * /)
-if condition
-print() function

**Task 2: Personalized Greeting**

*File name: task2_personalized_greeting.py*

**What this program does:**

-Takes the user's first name and last name
-Joins them to make a full name
-Prints a personalized greeting message

**Concepts Used:**

-String input
-String concatenation
-Variables
-Print statement

**How to Run the Programs:**

-Install Python 3.12.0
-Open Terminal / Command Prompt
-Run the program using:
-python task1_basic_math_operations.py
or
python task2_personalized_greeting.py
